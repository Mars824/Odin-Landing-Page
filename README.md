# Odin Landing Page

A responsive landing page built purely with HTML and CSS to advertise a fictional product.

# Getting Started 

Follow these steps to view the project locally.

1. CLone the repository to your computer.
2. Open the 'index.html' file in your web browser.

# Technologies Used

* HTML5 (for structure)
* CSS3 (for styling and layout)

  # Features

  * Clean, modern design.
  * Fully resonsive layout using FLexbox and Media Queries.
  * Clear Call-To-Action (CTA).
